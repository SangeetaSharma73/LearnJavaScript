# Javascript day1

## Let , Var and Const

1. Var =>

   - functional scoped/global
   - reassignment -> yes
   - redeclared with same variable-> yes
   - Hoisting ->yes (undefined)

2. Let =>
   - block scope,
   - reassignment -> yes
   - can't redeclare same variable
   - Hoisting -> Yes but in temporal dead zone
3. Const =>
   - block scope
   - No reassignment
   - Hoisting -> Yes but temporal dead zone
